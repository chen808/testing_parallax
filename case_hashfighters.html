Case Study: "Hashfighters"

Initial UI prototyping is fleshed out. This provided a blueprint on building the website and presented any potential issues to be aware of.

Utilizing Adobe Illustrator to construct a mockup based on the UI prototype. At this stage, the layout is becoming clear and the decision was made to prominently give the tweets more real estate because it is a community driven web app that encourages users to voice their opinions.

Design was heavily inspired by old boxing promotional posters. Features such as "Top 10 Fighters", "Vote for Upcoming Matches", and "Fighter Profile" were added.

Some of the issues encountered were how the incoming tweets affected the animation. Due to the unpredictable nature of using the Twitter Api, it was problematic in controlling the incoming tweets.

The initial solution was to 'bottleneck' the incoming tweets in Node to search for any repeats. Also the timer was increased to allow the animation to take effect.

This project was created using JavaScript, AngularJS, jQuery, HTML5, CSS, and Bootstrap.
UI prototyping, website mockup, and character animation was created with Adobe Illustrator and PhotoShop.
